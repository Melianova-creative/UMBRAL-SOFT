# UMBRAL-SOFT
Umbral Soft: solución tecnológica web orientada a automatizar y centralizar el agendamiento de citas, la gestión de clientes, el control de inventarios y el balance financiero de un salón de belleza independiente en Bogotá y sus alrededores. Proyecto académico del SENA ADSO, desarrollado durante el periodo educativo con metodología ágil Scrum.
